# POWER-BI-DASHBOARD
# 📊 Dashboard de Ventas Northwind

Este proyecto es un dashboard interactivo creado en Power BI usando el dataset Northwind. El objetivo es analizar ventas, tendencias y envíos de una forma visual y fácil de entender.

## 🛠 Herramientas
- Power BI
- Northwind Dataset
- GitHub

## 📈 Incluye
- KPI de ventas
- Ventas por año
- TOP 5 productos/clientes
- Mapa de países
- Filtros interactivos

Descarga el repositorio y abre el archivo `.pbix` en Power BI Desktop.
