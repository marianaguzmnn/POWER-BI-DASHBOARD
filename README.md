# POWER-BI-DASHBOARD
# 📊 Dashboard de Ventas Northwind

Dashboard interactivo desarrollado en Power BI utilizando el dataset Northwind para analizar ventas, tendencias y distribución geográfica de envíos.

## 🛠 Herramientas
- Power BI
- Northwind Dataset
- GitHub

## 📈 Características
- KPI de ventas totales
- Ventas por año
- TOP 5 productos/clientes
- Mapa de países de envío
- Filtros interactivos

## 🚀 Cómo usarlo
1. Descarga el repositorio.
2. Abre el archivo `.pbix` en Power BI Desktop.
