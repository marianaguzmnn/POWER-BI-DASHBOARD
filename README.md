# POWER-BI-DASHBOARD
#  Dashboard de Ventas Northwind
Este proyecto consiste en un dashboard interactivo desarrollado en Power BI utilizando el dataset Northwind. El objetivo es analizar el desempeño de ventas, identificar tendencias y visualizar información importante de manera clara y dinámica. El dashboard incluye KPIs de ventas, gráficos de tendencias, un TOP 5 de productos o clientes y un mapa de países de envío, además de filtros interactivos que permiten explorar los datos de forma más personalizada y obtener insights útiles para la toma de decisiones comerciales.
