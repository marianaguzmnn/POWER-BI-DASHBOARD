# POWER-BI-DASHBOARD
Dashboard interactivo en Power BI basado en el dataset Northwind. Analiza ventas, tendencias y distribución geográfica mediante KPIs, gráficos dinámicos y filtros interactivos para transformar datos en insights útiles para la toma de decisiones comerciales.
# 📊 Dashboard de Ventas Northwind | Power BI

## 📌 Descripción del Proyecto
Este proyecto consiste en un dashboard interactivo desarrollado en Power BI utilizando el dataset Northwind.  

El objetivo principal es analizar el desempeño de ventas, identificar tendencias comerciales y visualizar información geográfica relacionada con los envíos realizados por la empresa.

El dashboard permite transformar datos empresariales en información visual e interactiva para facilitar la toma de decisiones.

---

## 🎯 Objetivos
- Analizar el rendimiento total de ventas
- Identificar los productos/clientes con mejor desempeño
- Visualizar tendencias de ventas a lo largo del tiempo
- Analizar la distribución geográfica de envíos
- Crear un dashboard interactivo y dinámico

## 📈 Características del Dashboard

### 💰 KPI de Ventas Totales
Muestra el total de ventas generadas.

### 📅 Ventas por Año
Gráfico de líneas que permite visualizar el comportamiento de ventas por año y mes.

### 🏆 TOP 5 Productos/Clientes
Gráfico de barras con los productos o clientes con mayores ventas.

### 🌎 Mapa de Países de Envío
Visualización geográfica de los países de destino de los envíos.

### 🔍 Filtro por País
Permite filtrar dinámicamente toda la información del dashboard según el país seleccionado.
